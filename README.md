# Hospital Management System

A simple Java console-based mini project using MVC Architecture.

## Features
- Add Patient
- View Patients
- Add Doctor
- View Doctors
- Book Appointment
- View Appointments

## Technologies Used
- Java
- OOP Concepts
- MVC Architecture
- ArrayList
- Scanner Class

## Project Structure

HospitalManagement/
│
├── Patient.java
├── Doctor.java
├── Appointment.java
├── HospitalController.java
├── HospitalView.java
└── Main.java

## How to Run

1. Compile all files:
javac *.java

2. Run the project:
java Main
